# 🌾 Kisan Mitra — Smart Farming Dashboard

A modern, responsive **Smart Farming Dashboard** built for Indian farmers using pure HTML, CSS & JavaScript. Helps farmers make data-driven decisions about weather, soil health, crop management, pest alerts, and market prices.

## 🖥️ Live Demo
> Open `index.html` in any browser — no setup needed!

## ✨ Features

| Feature | Description |
|---|---|
| 🌦️ Weather Dashboard | 7-day forecast with farming advice in Hindi |
| 🌍 Soil Health Monitor | Real-time N, P, K levels, pH, moisture, temperature |
| 🌱 Crop Calendar | Track crops with growth stage badges |
| ⚠️ Pest & Disease Alerts | Priority-based alerts with treatment advice |
| 📈 Market Prices (Mandi Bhav) | Live-style crop prices vs MSP comparison |
| 💧 Irrigation Schedule | Week-wise smart irrigation planning |
| 🤖 AI Advisor | Contextual farming advice in Hindi |
| 📱 Fully Responsive | Works on mobile, tablet & desktop |
